### vim-file-types

File type plugin for vim.

### Install

Easy installing with [Vundle](https://github.com/gmarik/Vundle.vim)

Simply add to you plugins in .vimrc

    Bundle 'johnantoni/vim-file-types'

Launch VIM and run:
  
    :BundleInstall
